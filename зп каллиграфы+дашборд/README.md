# ЗП Каллиграфы + Дашборд — SignaturePro

Система расчёта зарплат каллиграфов и менеджеров на основе Google Таблицы с заказами.

---

## Состав проекта

| Файл | Назначение |
|---|---|
| `dashboard.py` | Flask-приложение (веб-дашборд) |
| `gsheets.py` | Адаптер Google Sheets API (чтение, перенос заказов) |
| `setup_google_sheet.py` | Разовая настройка таблицы (форматирование, формулы) |
| `config.py` | Настройки: URL таблицы, путь к credentials |
| `credentials.json` | Ключ сервисного аккаунта Google (не публиковать!) |
| `encode_credentials.py` | Утилита: кодирует credentials.json в Base64 для Render |
| `templates/index.html` | Интерфейс дашборда (HTML + Alpine.js + Tailwind) |
| `requirements.txt` | Зависимости Python |
| `Procfile` + `render.yaml` | Конфигурация деплоя на Render |
| `RENDER_CREDENTIALS_B64.txt` | Закодированный ключ — вставить в переменную окружения на Render |

## Структура Google Таблицы

Каждый лист = один месяц (напр. `Апрель_2026`).

| Столбец | Содержимое |
|---|---|
| A | Дата поступления |
| B | Статус (галочка = завершён) |
| C | Менеджер |
| D | Каллиграф |
| E | ФИО клиента |
| F | Тариф (E / ST / E_FAST / OPTNEW / PRNEW) |
| G | Дата: Варианты |
| H | Дата: ОС 1 |
| I | Дата: Правка 1 |
| J | Дата: ОС 2 |
| K | Дата: Правка 2 |
| L | Дата: ОС 3 |
| M | Дата: Правка 3 |
| N | Дата: ОС 4 |
| O | Дата: Правка 4 |
| P | Дата: Обучение |
| Q | Ссылка |
| R | Дата завершения |
| S | Заметки |
| T | Дата: ОС 5 |
| U | Дата: Правка 5 |
| V | Дата: ОС 6 |
| W | Дата: Правка 6 |
| X | **Итого ЗП** (формула, автоматически) |

## Логика начислений каллиграфу

| Событие | Условие начисления | E / ST | E_FAST | OPTNEW | PRNEW |
|---|---|---|---|---|---|
| Варианты | Заполнена дата G | 150 ₽ | 150 ₽ | 500 ₽ | 500 ₽ |
| Каждая правка | Заполнена дата правки (I/K/M/O/U/W) | 75 ₽ | 75 ₽ | 75 ₽ | 75 ₽ |
| Обучение | Заполнена дата P | 200 ₽ | 200 ₽ | 200 ₽ | 200 ₽ |
| Тарифный бонус | Варианты готовы (G заполнена) | — | 300 ₽ | — | 300 ₽ |
| Бонус без правок | B=✓, G заполнена, правок = 0 | 100 ₽ | 150 ₽ | 200 ₽ | 250 ₽ |

**Менеджеру** начисляется бонус за каждый завершённый заказ: E/ST/E_FAST = 200 ₽, OPTNEW/PRNEW = 300 ₽.

## Запуск локально

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Настроить доступ к Google Sheets

- Создать сервисный аккаунт в [Google Cloud Console](https://console.cloud.google.com/)
- Скачать JSON-ключ, сохранить как `credentials.json`
- Открыть доступ к таблице для email сервисного аккаунта (Редактор)
- Указать URL таблицы в `config.py`

### 3. Настроить таблицу (один раз)

```bash
python setup_google_sheet.py
```

Скрипт применяет к каждому листу:
- Условное форматирование (зелёный / жёлтый / серый по тарифу)
- Выпадающий календарь на ячейках с датами
- Формулу «Итого ЗП» в столбце X

### 4. Запустить дашборд

```bash
python dashboard.py
```

Открыть в браузере: [http://localhost:5000](http://localhost:5000)

## Деплой на Render

### Подготовка credentials

```bash
python encode_credentials.py
```

Скрипт выводит Base64-строку — скопировать в `RENDER_CREDENTIALS_B64.txt`.

### Переменные окружения на Render

| Переменная | Значение |
|---|---|
| `GOOGLE_SHEETS_URL` | URL Google Таблицы |
| `GOOGLE_CREDENTIALS_B64` | Содержимое `RENDER_CREDENTIALS_B64.txt` |

### Деплой

Подключить репозиторий GitHub к Render. При каждом `push` в `main` Render автоматически пересобирает и деплоит приложение.

## Перенос заказов между месяцами

В дашборде кнопка **«Перенести заказы»**:
- Переносит незавершённые заказы (без даты завершения) из одного листа в другой
- Сохраняет форматирование ячеек
- Автоматически применяет условное форматирование, календари и формулу «Итого ЗП» к целевому листу

---

## Обновление данных

Дашборд кеширует данные из Google Sheets (по умолчанию 45 секунд, настраивается в `config.py` через `CACHE_SECONDS`). Кнопка **«Обновить»** сбрасывает кеш принудительно.
